KTMM: TIERED MEMORY MODULE
==========================

The KTMM Project is a module for the Linux kernel designed to enhance the
current Clock algorithm for page replacement. The implemention is an alternate
approach to a dynamic tiering system for memory intensive applications on
modern server platforms.
